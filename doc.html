Documentation

For this project I needed to learn how to use some extra features of css and jQuery which were new to me.
These consisted mainly of Jquery animations, the filter feature of css, and @keyframes.
I also experamented with using svgs and pngs, in the end I used both for different purposes.
I also really worked a lot figuring out how to get images to stay together and scale uniformly to the page.
The project took about ten days to complete and was uploaded on the 31st of July.
